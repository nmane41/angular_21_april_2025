Object vs Map
==============

module
======
-Module Systems : Commonjs(nodejs) , EJS(Angular,React), AMD, UMD

CommonJs / CJS  : module.exports = {} , require()
EJS : import , export

Note : To load an ES module, 
    1. set "type": "module" in the package.json
    2. use the .mjs extension


Promise
=======
-A promise is an object whose value is not known now but will be available in near future.
-Promise is a pre-defined class in javascript
